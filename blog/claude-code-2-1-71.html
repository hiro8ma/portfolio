Claude Code 2.1.71 リリース — /loop コマンドで定期実行、Voice モードの安定性が大幅向上

Claude Code 2.1.71 がリリース

Claude Code の最新バージョン 2.1.71 がリリースされた。

今回の目玉は /loop コマンド。定期的にプロンプトを実行できるようになり、開発ワークフローの自動化がさらに進んだ。

新機能

/loop コマンド — 定期プロンプト実行

/loop 5m check the deploy

指定した間隔でプロンプトやスラッシュコマンドを繰り返し実行する。

ユースケース例:

  • デプロイの監視
  • テストの定期実行
  • ビルドステータスの確認
  • ログの監視

バックグラウンドで定期的にチェックしながら、メインの作業を続けられる。

Voice キーバインドのカスタマイズ

Push-to-Talk のキーバインドが keybindings.json でカスタマイズ可能に。

{
  "voice:pushToTalk": "ctrl+shift+v"
}

デフォルトはスペースキー。好みに合わせて変更できる。

Bash 自動承認リストの拡張

以下のコマンドが自動承認リストに追加：

コマンド用途
fmtテキスト整形
commファイル比較
cmpバイト比較
numfmt数値フォーマット
expr式評価
test条件テスト
printfフォーマット出力
getconfシステム設定取得
seq連番生成
tsortトポロジカルソート
prページング

これらのコマンドは確認なしで実行される。

バグ修正

stdin フリーズ問題の修正

問題: 長時間セッションでキー入力が処理されなくなる

これは深刻なバグだった。数時間作業していると突然キー入力を受け付けなくなり、セッションを再起動するしかなかった。

修正済み。

Voice モード起動時のフリーズ修正

問題: Voice モードユーザーで起動時に 5〜8 秒のフリーズ

原因は CoreAudio の初期化処理。非同期化されて解消。

OAuth トークンリフレッシュのフリーズ修正

問題: 複数の OAuth コネクタが同時にトークンをリフレッシュすると UI がフリーズ

MCP サーバーを複数使っているユーザーに影響していた。

/fork の計画ファイル共有問題

問題: /fork で会話を分岐しても同じ計画ファイルを共有してしまう

分岐した会話が独立した計画を持つようになった。

画像サイズの問題

問題: Read ツールが画像を過大にリサイズし、画像が多いセッションが壊れる

長い画像付きセッションでコンテキストが溢れる問題が解消。

Heredoc コミットメッセージの誤検出

問題: Heredoc でコミットメッセージを渡すと不要な権限確認が出る

git commit -m "$(cat <<'EOF'
commit message
EOF
)"

このパターンでの誤検出が修正された。

その他の修正

問題状態
プラグインインストールが複数インスタンスで失われる修正
Bash ツールのエラーメッセージが重複修正
Chrome 拡張の自動検出がスタック修正
--print がチームエージェント設定時にハング修正

改善点

起動時間の短縮

ネイティブ画像プロセッサのロードを遅延化。起動がより速くなった。

Bridge セッション再接続の改善

ラップトップのスリープ復帰後、数秒以内に再接続するようになった。

以前は再接続に時間がかかることがあった。

プラグインアンインストールの改善

/plugin uninstall が settings.json ではなく settings.local.json を変更するようになった。

グローバル設定を汚さない。

最近のバージョンの注目機能

v2.1.70 — VSCode 統合強化

  • スパークアイコン: アクティビティバーに全 Claude Code セッションを表示
  • プラン表示: Markdown ドキュメントビューでプランを確認
  • MCP サーバー管理: /mcp コマンドでネイティブダイアログ

10言語追加で計20言語をサポート：

  • ロシア語、ポーランド語、トルコ語、オランダ語
  • ウクライナ語、ギリシャ語、チェコ語
  • デンマーク語、スウェーデン語、ノルウェー語

v2.1.68 — Opus 4.6 デフォルト

  • Opus 4.6 が medium effort をデフォルト化（Max/Team）
  • ultrathink キーワードで high effort を有効化
  • Opus 4 / 4.1 は Opus 4.6 に自動移行

アップデート方法

npm update -g @anthropic-ai/claude-code

または：

claude update

まとめ

Claude Code 2.1.71 の主要な変更：

カテゴリ内容
新機能/loop コマンドで定期実行
カスタマイズVoice Push-to-Talk キーバインド
安定性stdin フリーズ、Voice 起動フリーズを修正
パフォーマンス起動時間短縮、Bridge 再接続改善

特に /loop コマンドは開発ワークフローを変える可能性がある。デプロイ監視やテスト実行を Claude に任せながら、別の作業に集中できる。
