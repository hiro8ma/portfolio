Claude Cowork — Claude Code のデスクトップ版

anthropicai-agentsclaudecowork

Cowork とは

Claude Code をデスクトップアプリで動かせるようにしたもの。Anthropic の公式説明は「Claude Code for the rest of your work」。

主な特徴

  • フォルダアクセス: 指定フォルダ内のファイルを読み書き可能
  • マルチステップ実行: タスクを分解して並列処理
  • プラグイン: 生産性、セールス、ファイナンスなど 11 種のプラグイン
  • ブラウザ連携: Chrome 拡張と組み合わせて Web 操作も可能

Claude Code との違い

| | Claude Code | Cowork | |---|---|---| | 環境 | ターミナル | デスクトップアプリ | | 対象 | 開発者 | 非エンジニアも | | OS | 全 OS | macOS のみ（現状） |

Claude Code ユーザーから見た感想

vibe coding で Claude Code を使ってた自分としては、AI がファイルを編集するのは既に日常。

Cowork は新体験というより、ターミナルを使わない人向けの入口。チームの非エンジニアに勧めるならこっち。
