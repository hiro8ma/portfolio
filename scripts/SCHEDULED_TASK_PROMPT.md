# Claude Code アップデート監視タスク

## 目的
Claude Code の新バージョンがリリースされたら、X投稿用のドラフトを作成して通知する。

## タスク手順

1. **バージョンチェック**
   ```bash
   npm view @anthropic-ai/claude-code version
   ```

2. **前回バージョンと比較**
   - ファイル `~/.claude-code-version` に前回のバージョンを保存
   - 同じなら「変更なし」と報告して終了

3. **新バージョンの場合**
   - CHANGELOG を取得して主要な変更点を抽出
   - 日本語で要約
   - X投稿用ドラフトを作成

4. **ドラフト形式**
   ```
   Claude Code v{version} がリリース 🎉

   主な更新:
   • {重要な変更1}
   • {重要な変更2}
   • {重要な変更3}

   #ClaudeCode #AI #開発ツール
   ```

5. **文字数制限**
   - X の140文字制限を意識
   - 簡潔かつ魅力的に

## 出力
- 変更がある場合：投稿ドラフトを明確に表示
- 変更がない場合：簡潔に報告
